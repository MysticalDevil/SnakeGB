# SnakeGB 极致体验打磨清单 (Aesthetic & Polish)

本文档整理了针对 SnakeGB v1.3.0 及后续版本的“工业级”打磨想法，旨在通过细微的感官增强，将游戏从“功能完备”提升至“极致沉浸”。

## 📳 1. 物理触感模拟 (Tactile Haptics)
*针对移动端（Android/iOS）触屏交互的补完。*
- [x] **按键震动 (Click Feedback)**：在 DPad 转向和按钮点击时触发极短的线性马达震动，模拟物理按键的触感。
- [x] **吞噬反馈**：吃到果子时触发轻微的“Pop”震动。
- [x] **死亡震荡**：撞墙死亡时触发强烈的低频震动，增强挫败感的视觉冲击力。

## 🎵 2. 音频动态交互 (Audio Dynamics)
*提升 8-bit 合成音色的灵动感。*
- [x] **环境声学滤镜 (LPF)**：当游戏暂停时，对 BGM 应用低通滤波（低沉化处理），营造“隔墙听音”的空间感。
- [x] **声场平移 (Panning)**：根据果子出现的 X 坐标，让吃食音效在左右声道产生 10%~20% 的偏移。
- [x] **动态混响**：随着蛇身变长，为 SFX 增加微弱的延迟反馈，模拟“成长”的厚重感。

## 📺 3. 视觉细节微调 (Visual Finesse)
*通过 Shader 进一步挖掘 LCD 物理特性。*
- [x] **陀螺仪驱动反射 (Dynamic Glare)**：利用手机加速度计，让屏幕上的“玻璃反光”随手机倾斜而微弱移动，产生惊人的物理立体感。
- [x] **像素电荷残留**：在蛇头经过的路径上，通过 Shader 模拟极短的“液晶分子复位延迟”，比目前的残影更细腻。
- [x] **边缘暗角 (Vignetting)**：模拟早期 LCD 屏幕边缘光损，增强复古照片感。

## 🕹️ 4. 交互逻辑优化 (QoL & Feedback)
- [ ] **动态按钮反馈**：当按下屏幕上的虚拟 DPad 时，对应的方向箭头发光或产生视觉下沉动画。
- [ ] **占位占比预警**：当蛇身占据超过 50% 棋盘时，背景网格随心跳节奏变色，提示空间局促。
- [ ] **无级音量体验**：优化侧边旋钮的阻尼感，增加滑动时的微弱震动。

## ✨ 5. 仪式感与趣味性 (Flavor & Juice)
- [ ] **经典开机序列**：模仿 GameBoy 开机动画，让标题从屏幕上方带有物理弹跳地落下，并伴随经典的“叮”声。
- [ ] **勋章奖励特效**：在勋章陈列室中，点击已获得的勋章会产生像素碎片喷发效果。

---
*注：本清单作为非功能性需求的参考蓝图，旨在追求艺术层面的极致表达。*
